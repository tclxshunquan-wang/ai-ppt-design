# AI PPT生成器 - 设计规范文档（Design Specification Document）

## 1. 设计概述

### 1.1 设计理念
**智能、高效、专业** - 通过AI技术降低PPT制作门槛，让每个人都能快速创作出美观专业的演示文稿。设计强调简洁直观的交互体验，减少用户认知负担，让用户专注于内容创作而非工具操作。

### 1.2 目标用户
- **主要用户**：产品经理、开发者、学生、商务人士
- **用户特征**：需要快速制作PPT，但不一定具备专业设计技能
- **使用场景**：工作汇报、产品介绍、技术分享、课程汇报、商业提案

### 1.3 设计原则
1. **简洁优先**：界面元素精简，只显示必要功能，避免信息过载
2. **实时反馈**：所有操作提供即时视觉反馈，让用户了解系统状态
3. **容错设计**：支持撤销、重做，提供确认对话框防止误操作
4. **一致性**：统一的视觉语言和交互模式，降低学习成本
5. **可访问性**：遵循WCAG 2.0标准，确保颜色对比度和可读性

## 2. 视觉设计规范

### 2.1 色彩系统

#### 主色调
- **主色（Primary）**：`#2563EB`（蓝色）- 体现科技感和专业性

  - 主色深色：`#1D4ED8`（用于active状态）
  - 主色超浅：`#EFF6FF`（用于背景高亮）

#### 辅助色彩
- **成功色（Success）**：`#10B981`（绿色）- 用于成功提示、完成状态
- **警告色（Warning）**：`#F59E0B`（橙色）- 用于警告提示、待处理状态
- **错误色（Error）**：`#EF4444`（红色）- 用于错误提示、删除操作
- **信息色（Info）**：`#06B6D4`（青色）- 用于信息提示、帮助说明

#### 中性色系统
- **文字颜色**：
  - 主要文字：`#111827`（深灰）
  - 次要文字：`#6B7280`（中灰）
  - 辅助文字：`#9CA3AF`（浅灰）
  - 禁用文字：`#D1D5DB`（极浅灰）
- **边框颜色**：
  - 默认边框：`#E5E7EB`
  - 聚焦边框：`#2563EB`（主色）
  - 错误边框：`#EF4444`（错误色）
- **背景颜色**：
  - 主背景：`#FFFFFF`（白色）
  - 次要背景：`#F9FAFB`（浅灰）
  - 悬浮背景：`#F3F4F6`（中浅灰）
  - 选中背景：`#EFF6FF`（主色超浅）

#### 功能色彩
- **圈选高亮**：`#3B82F6`（主色浅色，透明度20%）- 用于选中元素背景
- **圈选边框**：`#2563EB`（主色，2px实线）- 用于选中元素边框
- **拖拽激活**：`#059669`（深绿色）- 用于拖拽模式激活状态
- **加载动画**：`#2563EB`（主色）- 用于加载动画和进度条

### 2.2 字体规范

#### 字体族
- **中文字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif`
- **英文字体**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **等宽字体**：`"SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, "Courier New", monospace`（用于代码展示）

#### 字号系统
- **超大标题（H1）**：32px / 48px（字号/行高）- 用于Hero区域主标题
- **大标题（H2）**：24px / 36px - 用于页面标题
- **中标题（H3）**：20px / 30px - 用于区块标题
- **小标题（H4）**：18px / 27px - 用于卡片标题
- **正文（Body）**：16px / 24px - 用于正文内容
- **小字（Small）**：14px / 21px - 用于辅助文字、说明文字
- **极小字（Tiny）**：12px / 18px - 用于标签、时间戳

#### 字重
- **粗体（Bold）**：700 - 用于标题、强调文字
- **中等（Medium）**：500 - 用于按钮文字、重要信息
- **常规（Regular）**：400 - 用于正文、默认文字
- **细体（Light）**：300 - 用于大号标题（可选）

### 2.3 布局系统

#### 栅格系统
- **列数**：12列栅格系统
- **间距（Gutter）**：24px
- **边距（Margin）**：桌面端32px，移动端16px
- **容器最大宽度**：1440px（桌面端）

#### 间距规范
采用8px基础单位，确保视觉一致性：
- **XS（极小）**：4px - 元素内部微间距
- **S（小）**：8px - 图标与文字间距
- **M（中）**：16px - 卡片内边距、元素间距
- **L（大）**：24px - 区块间距、卡片间距
- **XL（超大）**：32px - 页面区块间距
- **XXL（特大）**：48px - 页面主要区块间距

#### 圆角规范
- **按钮圆角**：8px（主要按钮）、6px（次要按钮）、4px（小按钮）
- **卡片圆角**：12px（大卡片）、8px（小卡片）
- **输入框圆角**：8px
- **标签圆角**：16px（胶囊形）
- **头像圆角**：50%（圆形）

#### 阴影系统
- **Level 1（轻微）**：`0 1px 2px 0 rgba(0, 0, 0, 0.05)` - 用于输入框、标签
- **Level 2（小）**：`0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06)` - 用于卡片悬浮
- **Level 3（中）**：`0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)` - 用于下拉菜单、弹出层
- **Level 4（大）**：`0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)` - 用于模态框、属性面板
- **Level 5（特大）**：`0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04)` - 用于大型弹出层

## 3. 交互设计规范

### 3.1 导航系统

#### 主导航
- **位置**：顶部固定导航栏，高度60px
- **结构**：左侧Logo + 导航菜单（AI工作台、素材中心、历史项目、个人中心），右侧功能按钮（编辑/预览切换、导出）
- **交互**：
  - 当前页面高亮显示（主色背景或主色文字）
  - hover状态显示背景色变化
  - 点击跳转到对应页面
- **响应式**：移动端折叠为汉堡菜单

#### 页面跳转
- **跳转方式**：SPA单页应用，使用路由跳转，无页面刷新
- **过渡动画**：淡入淡出（fade），时长200ms
- **加载状态**：跳转时显示骨架屏或加载动画

#### 面包屑（如需要）
- **位置**：页面内容区域顶部
- **样式**：文字链接，分隔符使用 `/` 或 `>`
- **交互**：点击可跳转到对应层级

### 3.2 交互反馈

#### 按钮状态
- **默认状态**：主色背景，白色文字，圆角8px，阴影Level 1
- **悬停状态（Hover）**：背景色加深10%，阴影Level 2，光标变为pointer
- **点击状态（Active）**：背景色加深20%，阴影Level 1，轻微缩放（scale 0.98）
- **禁用状态（Disabled）**：背景色`#E5E7EB`，文字色`#9CA3AF`，无阴影，光标not-allowed
- **加载状态**：显示加载动画（spinner），按钮文字变为"处理中..."

#### 表单交互
- **输入框默认**：白色背景，边框`#E5E7EB`，圆角8px，内边距12px
- **输入框聚焦**：边框变为主色`#2563EB`，阴影Level 1（主色），外发光效果
- **输入框错误**：边框变为错误色`#EF4444`，显示错误提示文字
- **输入框禁用**：背景色`#F9FAFB`，文字色`#9CA3AF`，边框`#E5E7EB`
- **下拉框**：点击展开，显示选项列表，选中项高亮，hover显示背景色变化

#### 加载状态
- **页面加载**：显示骨架屏（Skeleton Screen），模拟内容结构
- **内容生成**：显示加载动画（spinner或进度条），文字提示"正在生成..."
- **实时预览**：逐页显示生成结果，每页淡入动画（fade in），时长300ms
- **文件上传**：显示进度条，百分比显示，完成后显示成功提示

#### 错误提示
- **位置**：输入框下方或操作区域顶部
- **样式**：红色文字`#EF4444`，小字号14px，图标+文字组合
- **交互**：自动显示，3秒后自动消失，或用户点击关闭按钮
- **类型**：
  - 内联错误：输入框下方显示
  - 全局错误：页面顶部横幅显示，可关闭
  - 确认对话框：删除等危险操作需要确认

### 3.3 动效规范

#### 过渡效果
- **页面切换**：淡入淡出（fade），时长200ms，缓动函数`ease-in-out`
- **元素出现**：淡入+轻微上移（fade + translateY），时长300ms
- **元素消失**：淡出+轻微下移（fade + translateY），时长200ms
- **面板展开/收起**：滑动动画（slide），时长300ms

#### 微交互
- **按钮点击**：轻微缩放（scale 0.98），时长100ms
- **卡片悬浮**：阴影加深，轻微上移（translateY -2px），时长200ms
- **输入框聚焦**：边框颜色变化+外发光，时长200ms
- **圈选元素**：边框和背景色同时出现，时长150ms
- **拖拽移动**：元素跟随鼠标，实时更新位置，无延迟

#### 加载动画
- **Spinner**：旋转动画，360度旋转，时长1s，无限循环
- **进度条**：从左到右填充，显示百分比
- **骨架屏**：闪烁动画（shimmer），模拟内容加载

- **次要按钮（Secondary）**：
  - 尺寸：高度40px，内边距左右16px，圆角8px

  - 尺寸：高度32px，内边距左右12px

  - 尺寸：32px × 32px，圆角6px
  - 样式：透明背景，hover显示背景色，图标居中

#### 输入组件

- **文件上传**：
  - 尺寸：高度120px，宽度自适应，圆角8px
  - 样式：虚线边框`#D1D5DB`，背景`#F9FAFB`，hover背景`#F3F4F6`

- **下拉选择框**：
  - 尺寸：高度40px，宽度自适应，内边距12px，圆角8px
  - 样式：白色背景，边框`#E5E7EB`，右侧下拉箭头图标

- **卡片（Card）**：
  - 尺寸：宽度自适应，内边距24px，圆角12px
  - 样式：白色背景，阴影Level 2，hover阴影Level 3
  - 使用场景：Features展示、案例展示、模板展示
- **标签（Tag）**：
  - 尺寸：高度28px，内边距左右12px，圆角16px（胶囊形）
  - 样式：主色超浅背景`#EFF6FF`，主色文字`#2563EB`，字号14px
  - 使用场景：快捷键标签、分类标签
- **列表（List）**：
  - 尺寸：项目高度48px，内边距左右16px
  - 样式：hover显示背景色`#F9FAFB`，选中显示主色超浅背景
  - 使用场景：历史对话列表、导航菜单

### 4.2 业务组件

#### 页面头部（Header）
- **高度**：60px
- **背景**：白色，底部边框1px `#E5E7EB`
- **内容**：
  - 左侧：Logo + 导航菜单（AI工作台、素材中心、历史项目、个人中心）
  - 右侧：编辑/预览切换按钮、导出按钮
- **固定定位**：顶部固定，滚动时保持可见

#### Chat区域组件
- **宽度**：40%（可拖动调整）
- **布局**：垂直布局，从上到下为历史对话区、快捷键标签区、输入框区
- **历史对话区**：
  - 高度：自适应，最大高度calc(100vh - 200px)，支持滚动
  - 样式：消息气泡式，用户消息右对齐（主色背景），AI消息左对齐（灰色背景）
  - 交互：支持横向滚动查看历史对话
- **快捷键标签区**：
  - 位置：输入框上方，横向排列
  - 内容：三个标签（一句话生成、大纲生成、描述生成）
  - 交互：点击自动填充提示词到输入框
- **输入框区**：
  - 位置：底部固定
  - 内容：多行输入框、上传按钮、发送按钮
  - 样式：白色背景，圆角8px，阴影Level 1

#### Preview区域组件
- **宽度**：60%（可拖动调整）
- **布局**：居中显示PPT画布，支持缩放
- **编辑模式**：
  - 画布：白色背景，居中显示，支持缩放（50%-200%）
  - 元素：可点击圈选，选中显示高亮边框和操作栏
  - 操作栏：悬浮在选中元素右上角，显示编辑、拖拽、添加到Chat、删除按钮
- **预览模式**：
  - 画布：PPT样式展示，元素不可编辑
  - 交互：支持翻页、全屏预览
- **生成状态**：
  - 显示加载动画和"正在生成..."文字
  - 逐页显示生成的PPT，每页淡入动画

#### 圈选操作栏组件
- **位置**：选中元素顶部右上角，悬浮显示
- **尺寸**：高度40px，宽度自适应（根据按钮数量）
- **样式**：白色背景，圆角8px，阴影Level 3，动画滑入效果
- **按钮**：
  - ✏️ 编辑：打开属性面板（仅单选）
  - ↔️ 拖拽移动：激活拖拽模式（仅单选，激活状态深绿色）
  - 💬 添加到Chat：将内容添加到输入框
  - 🗑️ 删除：删除元素（需确认）
- **多选状态**：显示选中数量提示，只显示添加到Chat和删除按钮

#### 属性面板组件
- **位置**：右侧悬浮，固定定位
- **宽度**：300px
- **布局**：垂直布局，从上到下为面板头部、内容编辑区、样式属性区
- **面板头部**：
  - 内容：标题"属性设置" + 收起按钮（✕）
  - 样式：底部边框1px `#E5E7EB`，内边距16px
- **内容编辑区**：
  - 文本元素：显示文本输入框，支持多行编辑
  - 图片元素：显示图片上传区域，支持上传、预览、更换、删除
- **样式属性区**（仅文本元素）：
  - 字体选择：下拉框
  - 字号选择：下拉框（12px-72px）
  - 颜色选择：颜色选择器
  - 对齐按钮：图标按钮组（左、中、右对齐）
  - 行距滑块：范围1.0-3.0，步长0.1
  - 字间距滑块：范围-2px-10px，步长1px
- **图片尺寸调整区**（仅图片元素）：
  - 宽度输入框：单位px，实时更新
  - 高度输入框：单位px，实时更新
  - 宽高比锁定：切换按钮，锁定状态下自动按比例调整
- **收起功能**：
  - 收起按钮：点击后面板向右滑出屏幕
  - 展开按钮：收起后右侧边缘显示展开按钮（蓝色背景，左箭头图标）

## 6. 响应式设计

### 6.1 断点设置
- **移动端**：< 768px
- **平板端**：768px - 1024px
- **桌面端**：> 1024px（优先设计）

### 6.2 布局适配

#### 桌面端（> 1024px）
- **首页**：单栏布局，Hero区域高度2/3，Features 4列网格
- **AI工作台**：两栏布局（Chat 40%，Preview 60%），支持拖动调整
- **素材中心**：两栏布局（导航栏固定宽度200px，素材列表自适应）

#### 平板端（768px - 1024px）
- **首页**：单栏布局，Hero区域高度1/2，Features 2列网格
- **AI工作台**：两栏布局（Chat 35%，Preview 65%），拖动调整范围缩小
- **素材中心**：导航栏可折叠，点击展开/收起

#### 移动端（< 768px）
- **首页**：单栏布局，Hero区域高度1/3，Features 1列列表，案例展示1列列表
- **AI工作台**：单栏布局，Chat和Preview切换显示（标签页切换）
- **素材中心**：导航栏折叠为底部标签栏，素材列表1列列表
- **Header**：导航菜单折叠为汉堡菜单

### 6.3 组件适配

#### 按钮组件
- **桌面端**：高度40px，内边距左右16px
- **移动端**：高度44px（符合触摸目标44×44px），内边距左右20px

#### 输入框组件
- **桌面端**：高度40px，字号16px
- **移动端**：高度44px，字号16px（防止iOS自动缩放）

#### 卡片组件
- **桌面端**：内边距24px，圆角12px
- **移动端**：内边距16px，圆角8px

#### 属性面板
- **桌面端**：右侧悬浮，宽度300px
- **移动端**：底部抽屉式，全宽，高度50vh，支持上拉展开/下拉收起

## 7. 开发交付说明

### 7.1 设计资产

#### 图标资源
- **图标库**：使用Heroicons或Lucide图标库（开源，MIT协议）
- **图标尺寸**：24×24px（桌面端），20×20px（移动端）
- **图标样式**：线性图标（outline），笔画2px，圆角外部1px
- **自定义图标**：PPT相关图标（如：生成、导出、编辑等）需要设计，SVG格式

#### 图片资源
- **Hero背景图**：可选，建议使用渐变背景或抽象图形，尺寸1920×1080px
- **Feature图标**：8个Feature图标，SVG格式，尺寸64×64px
- **案例缩略图**：PPT案例预览图，尺寸400×300px，JPG格式，压缩优化
- **占位图片**：图片上传占位符，尺寸120×120px，SVG格式

#### 字体资源
- **系统字体**：使用系统字体栈，无需额外加载
- **可选字体**：如需品牌字体，提供Web字体文件（woff2格式）

### 7.2 技术建议

#### 前端技术栈
- **框架**：React 18+ 或 Vue 3+（推荐React，生态更丰富）
- **UI组件库**：Ant Design 或 shadcn/ui（推荐shadcn/ui，更灵活）
- **状态管理**：Zustand 或 Redux Toolkit（轻量级推荐Zustand）
- **路由**：React Router 或 Vue Router
- **样式方案**：Tailwind CSS（推荐，快速开发）+ CSS Modules（组件样式）
- **动画库**：Framer Motion（React）或 Vue Transition（Vue）
- **文件处理**：File API + Canvas API（图片处理）

#### 开发工具
- **构建工具**：Vite（推荐，快速）或 Webpack
- **代码规范**：ESLint + Prettier
- **类型检查**：TypeScript（强烈推荐）
- **测试框架**：Vitest + React Testing Library

### 7.3 实现注意事项

#### 性能优化
1. **图片懒加载**：案例缩略图、模板缩略图使用懒加载，提升首屏加载速度
2. **虚拟滚动**：历史对话列表如果过长，使用虚拟滚动（react-window或vue-virtual-scroller）
3. **防抖节流**：输入框搜索、拖拽调整使用防抖/节流，减少计算频率
4. **代码分割**：按路由分割代码，实现按需加载
5. **缓存策略**：PPT数据使用IndexedDB存储，支持离线访问

#### 交互细节
1. **拖拽调整**：Chat和Preview区域拖动调整比例时，使用`resize`事件监听，实时更新布局
2. **圈选功能**：使用`pointer-events`控制元素可点击性，避免误触
3. **实时预览**：使用WebSocket或SSE实现实时更新，每生成一页立即显示
4. **撤销重做**：使用命令模式（Command Pattern）实现撤销/重做功能
5. **本地存储**：使用IndexedDB存储PPT数据，支持大数据量存储

#### 兼容性
1. **浏览器支持**：Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
2. **触摸支持**：移动端支持触摸操作，圈选功能支持触摸圈选
3. **键盘导航**：支持Tab键导航，Enter键确认，Esc键取消
4. **屏幕阅读器**：使用ARIA标签，支持屏幕阅读器访问

#### 错误处理
1. **网络错误**：显示错误提示，提供重试按钮
2. **文件上传错误**：文件过大、格式不支持时显示具体错误信息
3. **AI生成错误**：显示错误提示，允许用户重新生成
4. **数据保存错误**：自动重试3次，失败后提示用户手动保存

#### 安全考虑
1. **文件上传**：限制文件类型和大小，服务端验证文件内容
2. **XSS防护**：用户输入内容使用DOMPurify清理
3. **CSRF防护**：API请求使用CSRF Token
4. **数据加密**：敏感数据（如token余额）加密存储
